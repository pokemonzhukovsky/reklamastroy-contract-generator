# Генератор договоров РекламаСтрой

Предварительная версия локального веб-приложения для подготовки договора и
Приложения № 1 от ООО «Мастера Рекламы» или ИП Гассиева Алана Руслановича.

## Что уже работает

- выбор исполнителя: ООО с выделением НДС 22/122 или ИП без НДС;
- отдельные необязательные поля реквизитов заказчика;
- автоматическая подстановка даты приложения из даты договора;
- неограниченное количество позиций в таблице услуг;
- автоматический расчёт итога, НДС, предоплаты, остатка и сумм прописью;
- предоплата 70% по умолчанию с возможностью изменения;
- загрузка до трёх PNG/JPG изображений над таблицей приложения с сохранением пропорций;
- самостоятельная загрузка подписи и печати исполнителя без встроенных файлов по умолчанию;
- локальное сохранение загруженных подписи и печати в браузере на этом компьютере;
- опциональная подпись для ИП и ООО в Word/PDF, опциональная печать только для ООО в PDF;
- редактор полного текста каждого договора и поле дополнительных условий;
- живой предпросмотр договора и приложения;
- скачивание активного договора или приложения в Word без печати ООО (подпись — по выбору);
- прямое скачивание PDF с выбранной подписью и печатью;
- сохранение локального черновика в браузере.

## Запуск

Требуется Node.js 22.13 или новее и pnpm.

```bash
pnpm install
pnpm run dev
```

Откройте адрес, который появится в терминале (обычно
`http://localhost:3000`).

## Проверка

```bash
pnpm run build
pnpm run test
```

Сейчас это согласовательный прототип: перед рабочим использованием необходимо
утвердить формулировки договоров и правила расчёта НДС для цен ООО.
